# Técnicas de Programación – Ejemplos

Este repositorio contiene ejemplos simples en Python de las cuatro técnicas principales de programación orientada a objetos:

## Estructura del Proyecto
```
tecnicas_programacion/
 ├── abstraccion/
 │     └── ejemplo_abstraccion.py
 ├── encapsulacion/
 │     └── ejemplo_encapsulacion.py
 ├── herencia/
 │     └── ejemplo_herencia.py
 ├── polimorfismo/
 │     └── ejemplo_polimorfismo.py
 └── README.md
```

## Cómo ejecutar cada ejemplo

### Abstracción
```
cd abstraccion
python ejemplo_abstraccion.py
```

### Encapsulación
```
cd encapsulacion
python ejemplo_encapsulacion.py
```

### Herencia
```
cd herencia
python ejemplo_herencia.py
```

### Polimorfismo
```
cd polimorfismo
python ejemplo_polimorfismo.py
```

Cada archivo contiene comentarios explicativos para ayudarte a comprender la técnica aplicada.
